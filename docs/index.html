Node:

Nos permite correr JS en el servidor. Tambien nuestra pc puede ser un SV de desarrollo

Webpack

Es un empaquetador de modulos Nos ayuda a realiar trabajos de manera automatica (live reload, minimizar codigo, ofuscar, compatibilidad con otros navegadores)

  • Gestionar dependencias
  • Montar SV de desarrollo y prueba
  • Cargar modulos
  • Convertir diferentes formatos
  • Minimizar codigo
  • Compilar de SASS a CSS
  • Compilarde TS a JS
  • Nos permite trabajar con JS moderno

Crear proyecto de node

  • Node instalado: node --version
  • NPM instalado: npm --version
  • npm init
  • Crear el package,json

Instalacion y config de webpack

  • npm install webpack webpack-cli --save-dev
  • Se crea la carpeta dist
